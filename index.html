AI Unwrapped: Your Journey Through Artificial Intelligence

Welcome to the easiest place to learn AI.

Here, you will understand all the basics of AI theory and be guided through sample projects from Numpy and eventually PyTorch.

For this course I would recommend all code be done Google Colab; full project code can also be found here.

Find out more about the course below.

Google Colab

When following along with code, go to Google Colab. Then create a new notebook, you can connect to a Google computer which will run your code for you.

To add code you can hover your mouse in the center of the screen and click “+ code”, you can also set a shortcut in >tools >keyboard shortcuts.

To run code in an individual cell, click the triangle in the top left of the cell or press Ctrl + Enter.
